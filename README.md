### Usage 🤔

```
git clone https://github.com/imfunniee/gitark.git
```

```
Go to chrome://extensions
```

```
Enable Developer Mode
```

```
Click on "Load unpacked"
```

```
Choose the cloned "gitark" Folder
```

That's It, Enjoy 👀🔥

*works only on chrome 🤷‍♀️*
